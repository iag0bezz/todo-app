### 🎲 Executando o projeto
    
```bash
# Instalando as dependências
$ yarn install OR npm install

# Executar o projeto em modo de desenvolvimento
# O servidor será aberto na rota default http://localhost:5173/
$ yarn dev OR npm run dev
```
